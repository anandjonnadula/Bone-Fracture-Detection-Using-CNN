Training configuration
  • Transfer learning from ImageNet weights; two-phase training (frozen backbone, then fine-tuning of the top layers with BatchNorm kept frozen).
  • Class weighting to counter dataset imbalance{% if stage_key == 'stage1' %} (fractures are ~1:6.6 under-represented in training data){% endif %}.
  • X-ray-appropriate augmentation: horizontal flips, small rotations/shifts/zoom, brightness & contrast jitter.
  • EarlyStopping with best-weights restore; ReduceLROnPlateau schedule.
    • {% if stage_key == 'stage1' %}
  • Decision threshold tuned on a held-out validation split (not the test set).
    • {% endif %} {% if stage_key == 'stage2' %}
  • Label smoothing 0.1 for regularization on the small dataset.

Out-of-Distribution Gate

Non-radiographs (photos, documents, blank images) are refused before Stage 1 instead of receiving a confident fracture probability.

  • Layer 1 — color pre-filter: images with mean HSV saturation > 0.25 are rejected (radiographs are effectively monochrome).
  • Layer 2 — embedding distance: k-NN cosine distance (k = {{ ood_info.k }}) on Stage-1 backbone embeddings against {{ ood_info.refs }} reference training embeddings.
  • Acceptance threshold {{ ood_info.threshold }} = the {{ ood_info.percentile }}th percentile of training-set distances; synthetic negatives (blank/noise/gradient images) all score well outside it.
{% endif %}

Training Data

  • Stage 1: 5,345 training images (705 fracture / 4,640 no-fracture) and 594 held-out test images.
  • Stage 2: ~1,269 training images across 12 fracture-type folders (82–159 each) and 355 test images.
  • Images are public-source radiographs of mixed quality and provenance; labels follow the folder structure of the source dataset.

Known limitations

  • Trained on a small, web-scraped dataset — not on clinical PACS data; performance on real hospital images is unverified.
  • DICOM (.dcm) uploads are converted to 8-bit PNG with VOI LUT windowing; only non-identifying tags (body part, view, modality) are kept and the original file is deleted — no PHI is stored.
  • Grad-CAM shows model attention, which is not a clinically validated localization of the fracture.
  • The optional fracture-localization detector (when trained) is validated on pediatric wrist radiographs only; boxes on other body parts are unreliable and labeled "beta" everywhere they appear.
  • This is a student research project. It has no regulatory clearance of any kind and must not inform real medical decisions.
